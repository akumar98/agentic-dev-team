# Agentic Dev Team

A multi-agent software development framework built using CrewAI.

## Features

- Requirements Agent
- Architecture Agent
- Backend Agent
- Frontend Agent
- Testing Agent
- Review Agent

## Setup

```bash
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Create a `.env` file:

```
OPENAI_API_KEY=...
```

Run:

```bash
python main.py
```
